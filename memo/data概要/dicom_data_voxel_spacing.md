# data/dicom_data voxel spacing概要

集計日: 2026-06-17

## 集計対象

- 対象ディレクトリ: `data/dicom_data`
- サンプル数: 50
- シリーズ数: 50
- 読み取れたDICOMファイル数: 24,553
- 非DICOMまたはスキップしたファイル数: 1,496
- `PixelSpacing` 欠損ファイル数: 0

## 集計方法

- voxel spacingは `(row_spacing, col_spacing, z_spacing)` の順で記録した。
- `row_spacing` と `col_spacing` はDICOMの `PixelSpacing` から取得した。
- `z_spacing` は `ImagePositionPatient` のZ座標差分から算出した。
- Enhanced CTなど単一DICOM内に複数フレームがある場合は、per-frame `ImagePositionPatient` を使用した。
- 全50シリーズでZ方向spacingは `ImagePositionPatient` または per-frame `ImagePositionPatient` から算出できた。

## 要約

- 大半のシリーズは `z_spacing = 0.4 mm`。
- XY spacingは主に `0.351 mm`, `0.468 mm`, `0.546 mm` に集中している。
- 外れ値として `sample50` は `(0.253906, 0.253906, 1.497345)`、`sample9` は `(0.292969, 0.292969, 0.7)`。
- `SliceThickness` は `0.5 mm` や `1.0 mm` のケースがあるが、実際のZ間隔は `ImagePositionPatient` 差分を優先している。
