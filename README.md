# 頸椎領域別骨折検出および椎骨動脈損傷（VAI）リスク評価

## 📋 研究の背景と目的
頸椎外傷において、椎骨動脈損傷（Vertebral Artery Injury: VAI）の見落としは致命的である。VAIの診断における最大の障壁は、受傷直後は無症状である「Lucid Interval（空白期間）」の存在だ。この期間中に適切な処置がなされない場合、血栓症から後方循環系の脳梗塞へと進展し、死亡率は4%〜18%に達する。

**臨床的必要性とROI：** 先行研究によれば、この空白期間中に抗血栓療法を開始できれば、脳卒中発症率は約24%から数パーセントへと劇的に低減することが示されている。しかし、既存の骨折検出AIは「椎体単位」での検出に留まっており、臨床医が真に必要とする「血管損傷リスク（So What?）」を提示できていない。VAIリスクは、単なる骨折の有無ではなく「横突孔（椎骨動脈の通路）」への波及という解剖学的局在に極めて強く依存（相関：20-88%）するためである。

本プロジェクトの戦略的合理性は、椎骨を「解剖学的4領域」に高精度に分割し、リスク評価を自動化することにある。これにより、救急現場における「見落としゼロ」のスクリーニングを達成し、迅速なCTA（CT血管造影）移行への意思決定を強力に支援する。

- **対象**: 各椎体3DCTデータ(C1~C7)
- **データ形式**: DICOM -> NIFTI -> png

## システムアーキテクチャと全体パイプライン

入力データからリスク判定に至るまで、解剖学的整合性を数学的に保証しつつ、以下の4ステップを計画中。

### 1. **Preprocessing（前処理）**
- **Input:** 3D CT画像データ
- **Process:** TotalSegmentatorを用いて頸椎（C1-C7）を抽出し、個別椎体の3Dマスクを生成。
- **Output:** 椎体別3Dボリュームおよびマスク。

### 2. **Axis Correction（座標軸補正とスライス画像生成）**
- **Input:** 椎体別3Dボリューム。
- **Process:** **脊柱管（Spinal Canal）および左右の横突孔（Transverse Foramen）を幾何学的ランドマーク**として検出し、手動Re-slicingを実行。生理的前弯による傾きを排し、解剖学的軸に垂直な断面を再構成する。
- **Output:** 軸補正済み2Dスライス群。

### 3. **Region Segmentation（領域分割）**
- **Input:** 軸補正済み2Dスライス + 椎体マスク。
- **Process:** 独自開発の「ヒートマップ回帰モデル」を適用。
- **Output:** 4領域別の確率マップおよび境界線ヒートマップ。

### 4. **Fracture Detection and Risk Evaluation（骨折検出とリスク評価）**(計画段階)
- **Input:** 領域別マスク + 骨折検出結果。
- **Process:** 横突孔領域内の皮質断裂（Cortical Discontinuity）を検知。
- **Output:** VAIリスクの提示、およびCTA推奨アラートの出力。
