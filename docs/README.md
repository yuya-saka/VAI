# 研究進捗ドキュメント

このディレクトリは研究進捗の記録・管理に使用します。

## ディレクトリ構成

### `progress/`
研究の進捗記録を時系列で管理します。
- 週次・月次の進捗レポート
- マイルストーン達成記録
- 次のステップの計画

### `experiments/`
実験の設定・記録を管理します。
- 実験パラメータ
- 実験プロトコル
- データセット情報
- モデル設定

### `results/`
実験結果と分析を保存します。
- 定量的結果（精度、損失など）
- 定性的結果（可視化、サンプル出力）
- 比較分析
- 考察

### `notes/`
研究に関するメモ・アイデアを記録します。
- 論文メモ
- アイデア
- 技術調査
- 会議メモ

## 使い方

1. 新しい実験を開始する前に `experiments/` に実験設定を記録
2. 実験結果は `results/` に保存
3. 定期的に `progress/` を更新して進捗を追跡
4. 思いついたアイデアや調査内容は `notes/` に記録
