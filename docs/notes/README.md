# 研究ノート

研究に関するメモ・アイデア・調査内容を記録します。

## カテゴリ

### 論文メモ (`papers/`)
- 読んだ論文の要約
- 重要な手法・アイデア
- 引用情報

### アイデア (`ideas/`)
- 実験アイデア
- 改善案
- 仮説

### 技術調査 (`technical/`)
- ライブラリ調査
- 実装メモ
- トラブルシューティング

### 会議メモ (`meetings/`)
- ミーティング議事録
- フィードバック
- アクションアイテム

## テンプレート例

### 論文メモ
```markdown
# [Paper Title]

**Authors:**
**Published:**
**Link:**

## Summary


## Key Ideas


## Relevance to Our Work


## Notes
```

### アイデアメモ
```markdown
# Idea: <Title>

**Date:** YYYY-MM-DD
**Status:** Draft / Under Investigation / To Implement

## Background


## Proposal


## Expected Outcome


## Next Steps
```
