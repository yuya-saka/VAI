# Sigma Baseline 実験結果

**日付:** 2026-03-26
**実験名:** sigma 4パターン比較 (baseline)
**評価:** 5-fold cross validation

---

## 概要

ガウシアンヒートマップの sigma を 2.0 / 2.5 / 3.0 / 3.5 の4パターンで比較。
他の条件は固定（ALL vertebra, baseline pipeline）。

---

## 全体平均（5-fold average）

| sigma | MSE | Peak Dist (px) | Perp Dist (px) | Angle Error (°) | Rho Error (px) |
|-------|-----|----------------|----------------|-----------------|----------------|
| **2.0** | **0.001808** | **20.93** | 11.54 | 6.37 | 3.50 |
| 2.5 | 0.002020 | 22.59 | 11.43 | 6.54 | 3.64 |
| 3.0 | 0.002028 | 22.55 | 11.65 | 6.64 | 3.53 |
| **3.5** | 0.002009 | 22.23 | **11.40** | **5.74** | **3.38** |

- **sigma=2.0**: MSE・Peak Dist が最良（ピーク位置推定精度が最高）
- **sigma=3.5**: Angle Error・Rho Error・Perp Dist が最良（直線パラメータ精度が最高）
- **sigma=2.5, 3.0**: どちらの指標でも中間、優位性なし

### 部位別の傾向

- **C1・C2 が最も誤差が大きい**（7〜9°）。頸椎上位は解剖学的形状が不規則。
- **C3・C5 が最も小さい**（4〜6°）。中位頸椎は比較的安定。
- **C7** も大きめ（6〜8°）。
- sigma=3.5 は C1・C2・C4・C7 でベスト（上位・下位頸椎に有利）。
- sigma=2.0 は C3・C5 でベスト（中位頸椎に有利）。

## 考察

### トレードオフ

sigma を小さくする（2.0）と:
- ピーク位置精度が高い（Peak Dist 最小）
- MSE が低い
- ただし直線の角度推定が不安定になる傾向

sigma を大きくする（3.5）と:
- 直線パラメータ（角度・rho）の精度が向上
- 特に C1・C2 など形状が複雑な部位で効果
- ただしピーク位置精度が低下する

### 結論

- **総合ベスト → sigma=2.0**（ピーク検出精度が最優先ならば）
- **直線検出ベスト → sigma=3.5**（Angle Error: 5.74°, Rho Error: 3.38px）
- sigma=2.5, 3.0 は中間で優位性がなく、今後の実験から除外を検討

### 今後の方向性

- sigma=2.0 と 3.5 の中間点（例: 2.0 で訓練 → 直線パラメータのみ微調整）
- 椎骨レベルごとに sigma を変える（C1/C2: 3.5, C3/C5: 2.0）
- sigma とは別のアーキテクチャ改善（損失関数、データ拡張）

## 実験設定

- **モデル**: UNet (heatmap-moment ベース直線検出)
- **データ**: 全頸椎 (C1-C7), ALL splits
- **Folds**: 5-fold CV
- **変更点**: sigma のみ変更、他は固定 (baseline)
- **出力ディレクトリ**: `Unet/outputs/checkpoints_sig{X}_ALL_baseline/`
