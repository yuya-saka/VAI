## プロトタイプ：半平面分割（Half-Plane Partition）

アルゴリズム:
1. 各ポリラインにPCAフィット → 直線(法線nx,ny, 重心cx,cy)
2. ジャンクション検出: line_1-2, line_3-4 の最近接端点の中点
3. 法線方向の自動判定:
   - line_1: j_l（反対側junction）方向 = body側
   - line_3: j_r方向 = body側
   - line_2, line_4: 上方向(anterior) = body側
4. 各ピクセルを4本の半平面で分類:
   - body = line_1+ ∩ line_3+ ∩ line_2+ ∩ line_4+
   - right_foramen = line_1- ∩ line_2+
   - left_foramen = line_3- ∩ line_4+
   - posterior = 残り

**検証結果（全909スライス）:**

| 指標 | 現行実装 | 半平面分割 |
|------|---------|-----------|
| 成功率 | ~58% | **99.9%** (908/909) |
| fallback発動 | 80%+ | **不要** |
| ゼロ領域失敗 | 多数 | **1件 (0.1%)** |
| コード量 | 870行 | **~100行** |

領域サイズ比率:
- body: mean=27.7%, median=28.0%
- right_foramen: mean=16.3%, median=15.2%
- left_foramen: mean=15.3%, median=14.9%
- posterior: mean=40.7%, median=42.3%
