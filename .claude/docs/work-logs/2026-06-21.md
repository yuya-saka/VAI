## Stage1 分類器実装

### 概要

前処理済みデータ（2,012症例）を使って椎体レベル骨折分類のベースラインを実装。
RSNA 2022 1位解法 Type1（EfficientNetV2-S + BiLSTM）を `train_models/stage1/` に構築。

### 実装ファイル

```
train_models/stage1/
├── train.py
├── config/config.yaml
├── src/model.py, dataset.py, data_utils.py, evaluation.py, experiment.py, trainer.py
└── utils/losses.py, metrics.py
```

### 設計決定

- **データ分割**: 全体の20%を held-out test set として固定分離（学習・モデル選択に不使用）
  - train_val: 11,263 件 → 5-fold StratifiedGroupKFold（study単位）
  - test: 2,821 件 → 全fold best model のアンサンブルで最終評価
  - study overlap = 0、陽性率 train_val=10.2% / test=10.5% 確認済み
- **best model 基準**: `val_auroc`（最大化）。val_loss より最終評価と一貫性あり
- **fold 指定**: `config.yaml` の `start_fold` / `end_fold` で制御、CLI上書き可
