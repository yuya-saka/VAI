### アルゴリズム概要

1. 各ポリライン → PCAフィット → `FittedLine`
2. `j_r` = line_1/line_2 最近傍端点ペアの中点、`j_l` = line_3/line_4 同様
3. 各直線のbody側法線を決定:
   - line_1: j_l方向、line_3: j_r方向、line_2/line_4: 上方向（anterior）
4. 各椎骨ピクセルを符号付き距離で分類:
   - body = s1+ ∩ s3+ ∩ s2+ ∩ s4+
   - right_foramen = s1- ∩ s2+（rightが優先）
   - left_foramen = s3- ∩ s4+（rightと重複しない分）
   - posterior = 残り
5. 5ch one-hot マスク構築

## TotalSegmentatorマスク品質問題

### 問題

TotalSegmentatorが生成する椎骨マスク（outer boundary）が歪んでいる場合、  
領域マスクの品質も低下する。inner boundary（ライン注釈）は正確だが、  
outer maskへの依存が品質ボトルネックになっている。

### Codex推奨対策（優先順）

1. **QCフィルタリング**（工数低・即効）
   - 指標: `solidity`、隣接スライス連続性、line端点がmask内か、領域面積比率
   - 結果: `exclude / downweight / keep` に分類

2. **eroded maskベース化**（中期）
   - 椎骨マスクを少し収縮してから分割 → 外周歪みの影響を遮断
   - half-plane partitionのline主体設計はそのまま維持

3. Training時: QCスコアによるcurriculum learning（補助）

詳細: `codex/20260406-mask-quality-strategy.md`
