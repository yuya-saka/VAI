**3. 実装の主要ポイント**
- GT 直線の定義: アノテーション2点から法線形式 (φ, ρ) を計算
- 予測: ヒートマップ → モーメント → 主軸 → (φ, ρ)
- 符号合わせ: 法線ベクトルの内積で向きを統一
- 段階的学習:
  - Phase 1: ヒートマップ MSE のみ
  - Phase 2: + 角度損失（warmup後に重み増加）
  - Phase 3: + ρ損失（弱めに追加）
- 評価指標: 角度誤差、法線方向距離、GT線分からの平均垂線距離
